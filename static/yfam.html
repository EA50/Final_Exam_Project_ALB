The app allows management of multiple schedules that needs to be fit into one overall schedule without the hassle of double-checking and verification if there are any conflicting schedules. Thus, making multiple schedules into one easier and less time consuming.

- MS Excel documents are used to create the schedule.
- Any other document types will be ignored by the server.
- In case you have already clicked the Merge, Clear Uploads, or Clear Exports buttons, there will be no effect whatsoever on the server.
- Currently there is no support for custom tables. The template table is provided by clicking on the Download Template Table button.
- There may be instances where the Merge button causes the page to hang. A wrong table format may be used to upload. The correct table format is provided by the Download Template File button. The app can stil be used, however, it will not process that wrong table format. To reset, simply click the Clear Uploads button.


Download Template File BUtton
- Download the table template to be used to upload and merge.
Choose Files Button
- This is where you will provide the excel files where you want the schedules to be merged.
- A number can be given to the files in order of their significance. This is to set the order of priority for each table.
- The button will accept any file type but only the .xlsx file type will be used.
- Can upload up to 100 files. Any more than 100 files will result in a notification to reduce number of files to be uploaded.

Compile Files Button
- This will send the files in queue for merging.
- This does not merge the files, it simply 'packs' the files.
- After compiling the files, you can then merge the files through the Merge button.

Merge Button
- This will merge the files in queue for merging.
- After merging, you will be sent to a page where the merge is successful or not. The pages will also display the merged/invalid files to be downloaded.

Clear Uploads
- This will clear the files in the 'uploads' queue.
- After merging the files, this is a prerequisite for another merge of files as all the files in the uploads queue will be removed.

Clear Exports
- This will clear the files in the 'exports' queue.
- After merging the files, files in the exports queue will be stored in case for downloading the past merged files.

Summary Pages
- These are pages that will show the downloadable files. These files can either invalid schedules (or conflicting schedules) or merged files (or no conflicting schedules).
- There may be instances where downloadable files are not available because there are no such files.
- You may have accidentally clicked the Clear Exports button which is likely the cause.
